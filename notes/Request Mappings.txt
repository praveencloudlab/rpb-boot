Request Mappings
===========================

in the place of RequestMapping,


GET			:: listing
POST			:: posting new content
PUT			::	update full content
PATCH			:: update partial content
DELETE		:: delete 

 HTTP status codes
===========================
1x	:: 100	-  199 	- information
2x	::	200	-	299	- success
3x	:: 300	- 399 	- redirection
4x	::	400	- 499 	- client side error
5x	::	500	- 599 	- server side error
